# Network Scanning & Information Gathering using Nmap

## Objective

To perform network scanning and information gathering using Nmap and identify active hosts, open ports, running services, and operating system information.

## Tools Used

* Kali Linux
* Nmap
* VMware

## Steps Performed

1. Installed and verified Nmap.
2. Identified local network range.
3. Performed host discovery scan.
4. Conducted port scanning.
5. Performed service version detection.
6. Conducted OS detection.

## Results

* Active hosts were identified on the network.
* Open ports and services were discovered.
* Basic network mapping was completed.

## Learning Outcomes

* Network reconnaissance
* Port scanning techniques
* Service enumeration
* OS detection using Nmap
